ДЗ Подгона графиков

0. Для выполнения ДЗ2 нужна таже версия матлаба, что и для выполнения ДЗ1,
	включая доп. пакеты "Symbolic Math Toolbox" и "Control System Toolbox"

1. Для выполнения ДЗ2 и лабы 6 нужно скачать доп. пакет "Simulink Design Optimization"
	Скачать пакет: Home->Add-ons->"Get Add-Ons"

2. Скачай весь код из архива

3. Запуск 
	Открой файлы demon2.m и demon_sim.slx в папке src/demon_simulink получай результаты, оформляй отчёт
	Отчёт оформляй по примерам работ старшекуров
	https://drive.google.com/file/d/1sHPnUIrDFfNfwjr7jAUNW1QMpfP84i8K/view?usp=sharing
	https://docs.google.com/document/d/1hvCyU4ntI7h5q_E7bBjpu5ZjoYIeHKde/edit
	https://drive.google.com/file/d/1rfjgOUHBKUCvZZp1kLGmEwZuE18EnFMx/view?usp=sharing
	Помните, прогой пользуетсь не только вы. Можно словить бан за 60 одинаковых отчётов.
	Меняйте что-то в офорилении кроме фамилии.
